# ADPlay
ADPlay (ASCII-Drip play): Graphic content on your terminal

<p align="center">
  <img src="./demo.png">
</p>

## Build
Build bin and install dependencies: `cargo build --release`  
Bin location: `./target/release/adplay`

## Usage
Example: `adplay img.png`  
Need help ? Just use `adplay --help`
